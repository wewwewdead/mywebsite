About me

I'm John Mathew Loren, I'm a developer with a passion of creativity. I like to create something that interest me. I draw portraits using color pencils and I can also draw using Photoshop.

I started making artworks when I was 14 years old, it started when I joined facebook group called "Guhit Pinas, that time I've made 100 portraits in just a year.

I realized that my passion was creating something that interest me and share it to people.

Now I found a new hobby which is programming, learning how to program was fun, making concepts and solving bugs kinda make me feel alive. It started when I'm planning to buy a laptop. I asked my older brother that I'm planning to buy a laptop, and he suddenly replied to me to start learning how to program. He has already have a Udemy course called 100 days of code by Angela Yu.

Learning was fun the course is beginner friendly and python syntax is not that hard to grasp. After I learned the fundamentals of coding and become profecient in python, then I planned to create a social media app. I switched to javascript, html and css as my new programming langauge.
